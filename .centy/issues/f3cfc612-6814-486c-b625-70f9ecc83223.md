# Add cargo-husky and typos pre-commit hooks

Set up cargo-husky for git hooks and configure typos to check spelling on each commit. This keeps everything in the Rust ecosystem and helps maintain code quality by catching typos and spelling errors before they enter the codebase.

## Requirements

- Add cargo-husky as a dev dependency in Cargo.toml
- Configure pre-commit hook in Cargo.toml under [package.metadata.husky.hooks]
- Add typos-cli for spell checking (either as a build dependency or expect it installed)
- Create a _typos.toml or typos.toml config file for project-specific dictionary and exclusions if needed
- Ensure the pre-commit hook runs typos on staged files
- Document setup in README for contributors
