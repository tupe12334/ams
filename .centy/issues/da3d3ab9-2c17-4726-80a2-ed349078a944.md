# Add agent presets with –agent flag

Add –agent flag to ‘ams new’ to create sessions with specific AI agents pre-configured. Examples: ‘ams new my-project –agent claude’ creates session running claude, ‘ams new my-project –agent aider’ for aider, ‘ams new my-project –agent codex’ for codex. This is the core value prop of an Agents Manager Service.
